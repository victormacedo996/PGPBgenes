O que é BLAST?

O Basic Local Alignment Search Tool (BLAST) é um algoritmo para comparar informações de sequências biológicas primárias.

O programa compara sequências de consultas de nucleotídeos ou aminoácidos de DNA ou RNA a bancos de dados de sequências e aproxima diretamente alinhamentos que otimizam uma medida de similaridade local, a pontuação de par máximo de segmentos (MSP). Permite filtrar a saída, inferir relações funcionais e evolutivas entre sequências, bem como ajudar a identificar membros de famílias de genes. O BLAST também pode ser usado para realizar pesquisas de motivos, pesquisas de identificação de genes, analisar várias regiões de semelhança em sequências de DNA longas, remover regiões de baixa complexidade ou repetições de sequência na sequência de consulta, estender as correspondências exatas para pares de segmentos de alta pontuação (HSP) e avaliar a significância do escore HSP. Em suma, como o The New York Times chamou, é o Google da pesquisa biológica.

Sobre este projeto

O objetivo do projeto é capacitar pesquisadores que trabalham com biologia molecular para ter um acesso mais amigável, flexível e controlável sob a ferramenta NCBI Blast+.

Este programa é capaz de realizar o alinhamento de sequências de DNA contra um banco de dados contendo os genes acds, acka, esps, gdc,glta,mdh, nifh, phnx, phod, pita, ppc, pqq, psts. Para obter mais detalhes sobre como usar a ferramenta, acesse os tutoriais.
